cryogenでコメント付きブログを作ってgithub pagesでホストする

  1. leiningenをインストールする
  2. cryogen プロジェクトを作成する
  3. ローカル環境で lein ring server を実行する
  4. disqusアカウントを作成する
  5. https://disqus.com/admin/create/ で disqus short name を取得する
  6. http://cryogenweb.org/docs/configuration.html を参照して cryogen 上でdisqusを有効化する
  7. https://tangrammer.github.io/posts/02-12-2014-cryogen-and-github.html を参考にしてgithubにcryogenプロジェクトと生成したブログ・サイトをアップロードする
  8. github pageへのドメイン割当
